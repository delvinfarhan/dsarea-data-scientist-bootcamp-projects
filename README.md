# Python Data Analysis – DSArea Bootcamp

Repository ini berisi kumpulan tugas Python yang dikerjakan selama mengikuti **Bootcamp Data Science di DSArea (Digital Skills Area)**.

Tugas-tugas ini mencakup dasar pemrograman Python, pengolahan data menggunakan **Pandas**, serta **visualisasi data** menggunakan Matplotlib.

---

## Tentang DSArea

**DSArea (Digital Skills Area)** adalah lembaga pelatihan online yang berdiri sejak tahun 2021 dan menyediakan berbagai program pelatihan digital yang dibimbing oleh tutor berpengalaman. Program ini dapat diikuti oleh berbagai kalangan, baik pelajar maupun pekerja.

---

## Daftar Tugas

### 1. Dasar Pemrograman Python

Pada tugas ini dipelajari konsep dasar Python seperti:

- Menampilkan output dengan `print()`
- Pengecekan tipe data
- Konversi boolean
- Penggunaan variabel
- Operasi aritmatika
- String indexing dan slicing
- Manipulasi list

---

### 2. Analisis Data Menggunakan Pandas

Pada tugas ini dilakukan pengolahan data menggunakan library **Pandas**, meliputi:

- Membuat objek **DataFrame**
- Membaca dataset dari file **CSV**
- Membuat dan menggunakan **Series**
- Melihat data menggunakan `head()` dan `tail()`
- Melakukan analisis statistik deskriptif menggunakan `describe()`

**Dataset yang digunakan:**
- Diamonds Dataset

---

### 3. Analisis dan Visualisasi Data

Pada tugas ini dilakukan analisis data penjualan video game dengan teknik agregasi dan visualisasi data.

Beberapa langkah yang dilakukan:

- Mengelompokkan data menggunakan `groupby`
- Menghitung rata-rata penjualan berdasarkan genre
- Transformasi data menggunakan `melt` dan `pivot`
- Membuat visualisasi **Bar Chart**
- Membuat visualisasi **Stacked Bar Chart**

**Dataset yang digunakan:**
- Video Game Sales Dataset
